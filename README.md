# Habits app — Privacy Policy

This repository hosts the **Privacy Policy** for the Habits app on GitHub Pages.

**Live URL:** https://naga-siddhartha.github.io/Habit-Tracker-App/

This URL is used in the Habits app (Settings → Privacy Policy) and in App Store Connect.

## Enabling GitHub Pages

1. In this repo: **Settings → Pages**
2. Source: **Deploy from a branch**
3. Branch: **main**, folder: **/ (root)** → Save

The policy will be available at the URL above once Pages is enabled.
